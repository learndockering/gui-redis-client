# gui-redis-client
redis gui client: for CRUD

使用方法：解压出target目录中的zip文件，双击start.vbs即可运行（如果你的电脑不能运行，试试start.bat文件）。运行要求电脑中的PATH环境变量中有JAVA的bin路径，即java命令必须在PATH中。


+ JDK 1.7 Required
